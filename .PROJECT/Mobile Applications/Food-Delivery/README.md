# Food Delivery App

Aplikasi delivery makanan dengan tracking real-time dan pengalaman pengguna yang optimal.

## Fitur Utama
- Browse restaurants
- Menu & food items
- Shopping cart
- Multiple payment methods
- Real-time order tracking
- Delivery estimation
- Rating & reviews
- Promo & discounts

## Teknologi
- **Framework**: Flutter
- **State Management**: Riverpod
- **Backend**: Firebase
- **Maps**: Google Maps
- **Payment**: Stripe
- **Real-time**: WebSocket
- **Push Notifications**: FCM
- **Image Caching**: Cached Network Image

## User Roles
1. **Customer**: Order food, track delivery
2. **Restaurant**: Manage menu, orders
3. **Driver**: Accept orders, navigate
4. **Admin**: Manage platform

## Fitur Khusus
- Live location tracking
- Chat with driver
- Schedule orders
- Favorite restaurants
- Order history
- Loyalty points
- Multi-language support
