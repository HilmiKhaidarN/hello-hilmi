# Task Manager App

Aplikasi manajemen tugas dan produktivitas untuk meningkatkan efisiensi kerja.

## Fitur Utama
- Create, edit, delete tasks
- Task categories
- Priority levels
- Due dates & reminders
- Progress tracking
- Team collaboration
- Time tracking
- Reports & analytics

## Teknologi
- **Framework**: React Native
- **State Management**: Zustand
- **Database**: SQLite + Realm
- **Sync**: Firebase Firestore
- **Notifications**: React Native Push Notification
- **Calendar**: React Native Calendars
- **Charts**: Victory Native

## Fitur Lanjutan
- Pomodoro timer
- Habit tracking
- Goal setting
- File attachments
- Offline support
- Data export
- Dark mode

## Target Users
- Professionals
- Students
- Freelancers
- Small teams
