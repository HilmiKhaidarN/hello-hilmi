# E-Commerce Mobile App

Aplikasi e-commerce mobile untuk platform Android dan iOS.

## Fitur Utama
- Katalog produk
- Keranjang belanja
- Sistem pembayaran
- Tracking pesanan
- Profil pengguna
- Wishlist
- Review dan rating
- Push notifications

## Teknologi
- **Framework**: React Native
- **State Management**: Redux Toolkit
- **Navigation**: React Navigation
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Payment**: Stripe/Midtrans
- **Authentication**: Firebase Auth
