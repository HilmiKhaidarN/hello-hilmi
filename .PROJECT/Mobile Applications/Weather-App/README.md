# Weather App

Aplikasi cuaca modern dengan prediksi akurat dan interface yang menarik.

## Fitur Utama
- Cuaca real-time
- Prediksi 7 hari
- Cuaca per jam
- Multiple locations
- Weather alerts
- Beautiful animations
- Dark/Light theme
- Widget support

## Teknologi
- **Framework**: Flutter
- **State Management**: Bloc/Cubit
- **API**: OpenWeatherMap
- **Local Storage**: Hive
- **Notifications**: Flutter Local Notifications
- **Location**: Geolocator
- **Charts**: FL Chart

## Fitur Tambahan
- Radar cuaca
- UV Index
- Kualitas udara
- Sunrise/Sunset
- Humidity & Pressure
- Wind speed & direction
