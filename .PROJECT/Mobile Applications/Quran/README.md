# Al-Quran Mobile App

Aplikasi Al-Quran digital yang komprehensif dengan fitur audio, terjemahan, dan pembelajaran.

## Fitur Utama
- Teks Al-Quran lengkap 30 Juz
- Audio recitation (multiple qari)
- Terjemahan Bahasa Indonesia
- Tafsir Al-Quran
- Bookmark ayat favorit
- Pencarian ayat
- Jadwal sholat
- Kiblat direction
- Dzikir & doa harian

## Teknologi
- **Framework**: React Native / Flutter
- **Audio**: React Native Sound / AudioPlayers
- **Database**: SQLite
- **Location**: Geolocation
- **Compass**: React Native Sensors
- **Offline**: AsyncStorage
- **Typography**: Arabic fonts

## Fitur Tambahan
- Night mode
- Font size adjustment
- Reading progress
- Daily verse notification
- Tajweed highlighting
- Memorization helper
- Prayer time alerts

## Content
- Al-Quran text (Uthmani script)
- Audio files (MP3)
- Indonesian translation
- Tafsir Ibn Kathir
- Asmaul Husna
- Daily supplications
