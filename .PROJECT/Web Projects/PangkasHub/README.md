## üìä Database Schema

### Main Tables:
- **users**: Customer and admin accounts
- **appointments**: Booking records
- **services**: Available services and pricing
- **barbers**: Staff information and schedules
- **time_slots**: Available booking times

### Relationships:
- Users can have multiple appointments
- Appointments linked to services and barbers
- Time slots managed per barber
